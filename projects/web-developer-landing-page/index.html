For studying, you will need a computer with an internet connection.

About the Course

This course will introduce you to modern JavaScript-based web development. The main focus is on building single page applications with ReactJS that use REST APIs built with Node.js. On other courses you will learn to use various tools and technologies.

The studies include the following parts:

  • Programming
    • HTML5 and CSS3
    • JavaScript programming
    • UI design and prototyping
  • Working as a software developer
    • SQL and data structures
    • Basics of testing
    • Software development project 2
    • React basics
    • React extension
    • PHP programming
  • Implementing software with a software component library
    • WordPress basics
    • Software development project 1

The studies proceed in an interleaved manner, not in chronological order according to the degree components.
